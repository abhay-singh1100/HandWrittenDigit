72|normal|Predicting whether a customer will complete a purchase during a session has become a critical task  for businesses. Accurate purchase predictions help companies optimize user experience, personalize  recommendations, prioritize high-intent users, and allocate marketing resources more effectively.  W
73|normal|This project uses session-level data combined with machine learning techniques to build a predictive  model that estimates the likelihood of purchase. The solution also includes deployment, monitoring,  and explainability aspects, making it suitable for production environments.
74|normal|1.1 Problem Statement
75|normal|Despite having large amounts of user interaction data, many e-commerce platforms lack an automated  mechanism to analyze user behaviour in real time and predict whether a visitor will convert. Manual  analysis or static rule-based systems often fail due to:
76|normal|• Complex non-linear patterns in user behaviour
77|normal|• Highly imbalanced data where only a small percentage of sessions result in purchases • Lack of real-time scoring systems
78|normal|• Absence of explainability for predictions
79|normal|Therefore, the core problem addressed in this project is:
80|normal|To build a machine learning system that can accurately predict the probability of purchase for  each web session using user behaviour data, and deploy it for real-time inference.

135|normal|3.1.5 No Deployment or Monitoring Setup
